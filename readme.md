##  项目概览

这里是我的数字花园，用于记录技术学习笔记、生活随笔与赛博空间中的碎片思考。

**核心特性：**
-  **动态星空背景**：纯 CSS 实现的轻量级粒子动画，告别 JS 性能焦虑。
-  **终端控制台 UI**：About 页面采用伪终端设计，沉浸式极客体验。
-  **霓虹光晕与玻璃拟态**：现代 CSS 视觉特效，丝滑的交互动画。
-  **Zero Dependencies**：零依赖，零构建工具，加载速度拉满。
